# 个人学术主页 1:1 国科大官方风格重构设计文档

本文档详细记录了针对个人学术主页 `konata3b.github.io` 的重构设计，旨在以 1:1 像素级复刻中国科学院大学（UCAS）教师门户主页的经典风格，同时承载个人基本学术信息与中英双语切换功能。

## 1. 背景与目标
用户希望将 `konata3b.github.io` 目录下的 `index.html` 替换为经典的国科大教师主页样式：
*   **1:1 完美复刻**：直接采用从国科大官方系统提取的三个底层核心样式表（`base.css`、`myzone.css`、`homePage.css`），彻底确保边框、行高、菜单位置、字体类型等在像素级上与官方保持 100% 一致。
*   **保留个人信息**：内容填充为原主页中 Konata 的教育背景、关于我、研究兴趣与竞赛活动。
*   **图片资产更新**：头部加载目录下的最新 `logo.png`，个人基本信息加载目录下的 `photo.jpg` 证件照。
*   **中英双语平滑切换**：设计轻量级客户端 JS 方案，支持用户在右上角自由、平滑（带淡入淡出动效）地切换中/英文。

---

## 2. 详细设计规格

### A. 整体架构与布局 (DOM Structure)
基于官方经典的 `myzone3` 双栏排版模板：
*   `#header`：白色背景顶栏，底边为 3px 官方深院徽蓝（`#003D79`）实线。
    *   左侧 `#logo` 加载 `logo.png`，宽限制在 `486px`，高 `100px` 内，自适应垂直居中。
    *   右侧 `.b-topbar` 包含中英文切换链接 `[中文]` 与 `[English]`。
*   `#main-content`：核心内容容器，定宽 `940px`，水平居中，边框为官方经典的 `1px solid #ccc`。
    *   右侧 `.b-menu`（ sidebar）：宽度 `200px`，向右浮动（`float: right`），外边距 `20px`，边框 `1px solid #ccc`。内部为 list-style-type 为 `disc` 的导航菜单，链接通过 hash 锚点定位。
    *   左侧 `.b-pinfo`（基本信息卡片）：与菜单同级，使用 `overflow: hidden` 建立独立的 BFC 避开右浮动菜单。
    *   左侧 `.m-itme`（内容板块，注意类名为官方原版的 typo `m-itme`）：包含关于我、教育背景、研究兴趣、竞赛与活动，均作为同级块级元素在左侧流式向下排列。

### B. 视觉控制与对齐优化 (Aesthetics & Alignment)
*   **名片（.bp-enty）高度对齐**：个人证件照 `photo.jpg` 在左侧浮动，尺寸定为官方精确的 `200px` 宽、`230px` 高。为了确保文字内容不发生折行溢出或破坏与照片的垂直对齐，**名片信息区的文字字号保持官方原汁原味的 `12px` 不做放大**，行高 `1.8`，这完美地使姓名、身份、邮箱、地址与照片右侧高度平行对齐。
*   **正文清晰度提升**：关于我、教育背景及竞赛活动列表等主要阅读区域 of 字号，在官方样式的基础上重载提升至 **`15px` / `16px`**，并优化行高（`1.8`），以保证在现代高分屏下具备极高的阅读舒适性。
*   **就读院校与专业简化**：根据用户反馈，在“教育背景”板块中删除了“就读专业”及“就读院校”的二级子标题，直接呈现“中国科学院大学 (2026–2029)”，界面更加简洁。
*   **研究兴趣定制**：研究兴趣气泡标签（Tag）精准定制为：“大语言模型”（Large Language Models）、“多模态模型”（Multi-modal Models）与“智能体”（Intelligent Agents）。
*   **竞赛成果格式化占位**：清空了原有的获奖文字内容，并完整保留了官方列表项容器、数字点前缀与 `font` 格式，确保用户后期可以通过源码轻松直接填充自己的真实竞赛活动。

### C. 多语言支持 (Internationalization)
*   所有需要国际化的文本元素在 HTML 中采用 `data-zh` and `data-en` 进行声明式定义。
*   编写轻量高效的 JS 函数 `setLanguage(lang)`：
    1.  遍历所有带有 `data-zh` 属性的 DOM 元素。
    2.  触发快速的淡出淡入动画（`opacity` 从 `0` 渐变为 `1`），确保视觉切换非常自然高级。
    3.  根据当前选择，动态更新页面文本，并将用户的语言偏好以 `lang-preference` 键名持久化写入浏览器的 `localStorage` 中。

## 3. 验证方案 (Verification Plan)
1.  **本地服务运行与验证**：使用预览服务器提供服务，在 Chrome/Edge 浏览器中对 `layout-replica-v2.html` 进行高保真校验。
2.  **像素级对齐校验**：
    *   检查 Logo 图像在顶部的缩放与位置。
    *   检查照片 `photo.jpg` 与名片文字 `.bp-enty` 之间的垂直高度是否完美对齐。
    *   检查「竞赛与活动」板块下的列表格式，确认文本已完全清空，且序号排版无缺失。
    *   检查「教育背景」下是否已完全悬挂移除「就读院校」及「就读专业」二级子标题，仅直接显示大学名称和年份。
3.  **多语言点击测试**：点击 `[中文]` 和 `[English]`，验证文本是否瞬间优雅转换，并且刷新页面后能正确记住上次的选择。
4.  **移动端响应式测试**：缩窄浏览器窗口，验证流式布局在各尺寸屏幕下的排列逻辑。
